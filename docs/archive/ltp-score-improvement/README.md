# Archived LTP score-improvement artifacts

This directory stores historical LTP score-improvement sprint artifacts that are not part of the current active working set.

Archived content is preserved for audit and evidence lookup. Most files keep their original text, including old path strings such as `docs/ltp-score-improvement-YYYY-MM-DD-phase-x/...`; when following those historical references from the current tree, prefix them with `docs/archive/ltp-score-improvement/`.

Current or future active sprint directories should stay at the top level as `docs/ltp-score-improvement-YYYY-MM-DD-phase-x/` until they are no longer part of the active workflow.
